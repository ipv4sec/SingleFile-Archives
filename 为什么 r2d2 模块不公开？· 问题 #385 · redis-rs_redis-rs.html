提供的功能通过模块rd2d-redis内置到crate 中。下面是池连接的简单示例：redisr2d2

use redis::Commands;

fn main() {
    // get redis client
    let client: redis::Client = redis::Client::open("redis://127.0.0.1/").unwrap();

    // create r2d2 pool
    let pool: r2d2::Pool<redis::Client> = r2d2::Pool::builder().build(client).unwrap();

    // get a connection from the pool
    let mut conn: r2d2::PooledConnection<redis::Client> = pool.get().unwrap();

    // use the connection as usual
    let _: () = conn.set("KEY", "VALUE").unwrap();
    let val: String = conn.get("KEY").unwrap();
    println!("{}", val)
}

确保r2d2在你的中包含该功能Cargo.toml：

[dependencies]
redis = { version = "0.17.0", features = ['r2d2'] }
r2d2 = "0.8.4"
